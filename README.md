hotreload
=========

Add hot reload on static pages.

Copy & Paste this:

```html
<script src="https://cdn.jsdelivr.net/gh/tom-010/hotreload/hotreload.js"></script>
```

```html
<html>

<body>
    <h1>Test</h1>

    <script src="https://cdn.jsdelivr.net/gh/tom-010/hotreload/hotreload.js"></script>
</body>

</html>
```
